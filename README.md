# SoundCloud Likes Downloader

Скрипт для скачивания лайкнутых треков с SoundCloud.

## Возможности

- Скачивание всех треков из плейлиста лайков SoundCloud
- Автоматическая конвертация в MP3 (если установлен FFmpeg)
- Параллельная загрузка нескольких треков
- Пропуск уже скачанных файлов
- Автоматическое определение превью и их пропуск
- Обработка ошибок и повторные попытки при сбоях

## Требования

- Python 3.7+
- yt-dlp
- FFmpeg (опционально, для конвертации в MP3)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. (Опционально) Установите FFmpeg для конвертации в MP3:
   - Windows: Скачайте с [ffmpeg.org](https://ffmpeg.org/download.html) и добавьте в PATH
   - Linux: `sudo apt install ffmpeg`
   - macOS: `brew install ffmpeg`

## Использование

### Базовое использование

```bash
python main.py <URL>
```

### С указанием папки для сохранения

```bash
python main.py <URL> downloads
```

### С указанием количества параллельных загрузок

```bash
python main.py <URL> downloads 5
```

### Примеры

```bash
# Скачать лайки пользователя в папку downloads
python main.py https://soundcloud.com/user/likes

# Скачать в папку music с 10 параллельными загрузками
python main.py https://soundcloud.com/user/likes music 10
```

## Параметры

1. **URL** (обязательный) - Ссылка на плейлист лайков SoundCloud
   - Формат: `https://soundcloud.com/username/likes`
   
2. **output_dir** (опционально) - Папка для сохранения файлов
   - По умолчанию: `downloads`

3. **max_workers** (опционально) - Количество параллельных загрузок
   - По умолчанию: `1`
   - Рекомендуется: 3-10 (зависит от скорости интернета)

## Особенности

- **Автоматическое определение формата**: Если FFmpeg установлен, файлы конвертируются в MP3 320 kbps
- **Пропуск превью**: Треки короче 60 секунд автоматически пропускаются
- **Проверка целостности**: Файлы проверяются на полноту перед сохранением
- **Умная проверка дубликатов**: Сравнение по нормализованным названиям файлов

## Примечания

- Скрипт автоматически пропускает уже скачанные треки
- При ошибках выполняется до 3 попыток скачивания
- При ошибке 429 (слишком много запросов) скрипт автоматически ждет и повторяет попытку
- Гео-ограниченные треки будут пропущены с соответствующим сообщением
